# analise-superstore-sql

Projeto de análise de dados utilizando SQL para explorar padrões de faturamento, lucro e impacto de descontos.

---

## Objetivo

Analisar:
- Faturamento  
- Lucro  
- Desconto  
- Volume de vendas  

## Principais Insights

- A região West apresenta o maior faturamento e lucro
- Regiões com maior média de desconto tendem a ter menor lucratividade
- Descontos elevados (como 80%) podem gerar prejuízo
- A relação entre desconto e volume de vendas não é direta

## Ferramentas utilizadas

- SQL (MySQL)  
- phpMyAdmin  
- Python (pandas)
